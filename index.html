FlyPredictome Network Viewer

Interactive exploration of the Drosophila protein-protein interaction atlas.

FlyPredictome Database → — search any protein pair for predicted structures, PAE heatmaps, and the iLIS metric.

Network Construction

This network shows evidence-supported positive PPIs from the FlyPredictome atlas:

  1. Drosophila protein pairs were scored with AlphaFold-Multimer and iLIS
  2. Positive PPIs were selected at iLIS ≥ 0.223 (10% FPR)
  3. Only PPIs with prior evidence were retained: fly literature, BioGRID, genetic interaction (MIST), or conserved interolog (DIOPT ≥ 4)
  4. Proteins with ≥ 2 connections were kept (degree ≥ 2 filter)
  5. The giant connected component was extracted

Evidence Filter

  • All — All evidence types combined
  • Fly only — PPIs supported by Drosophila evidence (literature, BioGRID, GI) but not conserved interologs
  • Interolog only — Conserved interologs (DIOPT ≥ 4) without fly evidence
  • Both — PPIs supported by both fly evidence and conserved interologs — highest confidence

Edge colors: blue = fly evidence, teal = both, orange = interolog only.

FPR Filter

  • ≤10% — All evidence-supported PPIs (iLIS ≥ 0.223)
  • ≤5% — Higher-confidence PPIs (iLIS ≥ 0.339)
  • ≤1% — Highest-confidence PPIs (iLIS ≥ 0.551)
